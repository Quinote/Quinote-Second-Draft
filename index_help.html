User Guide

  1. Create an account and login
  2. When on the internal home screen, enter a title in the text box -> click the make new file button.
  3. This will take you to the QuiNote text editor. QuiNote understands your notes through placing of line breaks, colons, and semi-colons:
    • term 1: definition
    • term 2: definition 1; optional definition 2; optional definition 3
    • You can check out an example note file HERE Try copy-pasting that file if you want to see what quizzes look like right away
  4. To generate a quiz, enter what number of each type of question you'd like in the floating sidebar and click Generate Quiz

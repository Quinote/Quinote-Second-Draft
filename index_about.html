The Name

QuiNote is a portmanteau of the words quinoa and note (two things we're passionate about!).

The Project

QuiNote is our group's project for Mike Siff's spring 2015 Software Development course at Sarah Lawrence College. Tasked with creating a large-scale software application, we came up with this idea for a product to make future students' lives easier.

The Product

QuiNote is a self study aid that scans your notes in order to generate multiple choice, true/false, and fill-in-the-blank quizzes. QuiNote features a file manager (in which you can create, manage, and save files), a text editor and a quiz generator.


QuiNote is open source! Take a look at our code on Github
